# WPoets Full Stack Developer Test

## Project Overview

This project is a dynamic CRUD-based slider application developed using PHP, MySQL, HTML5, CSS3, jQuery, Bootstrap, and Slick Slider.

The application allows administrators to manage categories and slides dynamically from the admin panel. The frontend displays synchronized content and image sliders with responsive behavior for desktop and mobile devices.

---

## Features

### Admin Panel

* Category CRUD
* Slides CRUD
* Image Upload Functionality
* Category-wise Slide Mapping

### Frontend

* Dynamic Tabs
* Synchronized Content & Image Sliders
* Responsive Mobile Accordion
* Background Image Sliders on Mobile
* Fully Dynamic Data Rendering from Database

## Installation Steps

### 1. Clone Repository

```bash
git clone <your-repository-url>
```

### 2. Move Project

Place project inside:

```txt
xampp/htdocs/
```

### 3. Create Database

Open phpMyAdmin and create database:

```txt
wpoets_test
```

### 4. Import SQL

Run the SQL queries from:

```txt
schema.sql
```

### 5. Start XAMPP

Start:

* Apache
* MySQL

### 6. Run Project

Frontend:

```txt
http://localhost/full-stack-test
```

Admin:

```txt
http://localhost/full-stack-test/admin/categories.php
```
